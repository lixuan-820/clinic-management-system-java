# Clinic Management System (Java)

## Overview

This project is a console-based Clinic Management System developed in Java using object-oriented programming (OOP) principles.

The system allows clinic staff to manage patients, appointments, and basic medical records through a structured and menu-driven interface.

The project demonstrates Java class design, object modeling, data handling, and modular program structure.
